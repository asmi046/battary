# Создание скрипта импорта товаров

Нужно реализовать масовое добавление товаров с изображениями.
Исходные данные берутся из загружаемого excel файла.
Изображения заргужаются так вместе с файлом.

Компоненты для загрузки размещаем в вюшке resources/views/import.blade.php

Алгоритм работы:

- Пользователь загружает excel файл (формат входного файла опишу ниже) и набор изображений (имя файла изображения будет соответствовать полю 'Код' во входном excel файле)
- скрипт разбирает excel файл и добавляет товары в базу данных. Товары описываются моделью app/Models/Product.php
- по окончании работы скрипт выводит отчкет

## Формат входного excel файла:

- разбор начинаем с ячейки B7
- в 7й строке описана шапка
- в столбце С находится Полное наименование - его помещаем в наименование товара (name)
- в столбце D находится Код - его помещаем в поле sku
- в столбце E находится Полярность - ее помещаем в поле clem_location по следующему правилу:
    - обратная - 0
    - прямая - 1
- в столбце F находится Бренд - его помещаем в поля brand и series
- в столбце G находится Ток разряда - его помещаем в поле amperage
- в столбце I находится габариты - формат во входящем файле 352/175/190 (дл/шир/выс) разносим по полям поле height, width, length
- в столбце J находится емкость - ее помещаем в поле volume
- в столбце J находится напряжение - его помещаем в поле voltage
- в столбце M находится Тип - его помещаем в поле type по следующему принципу:
    - Аккумулятор -> Аккумуляторы для легковых автомобилей

## Загрузка изображений

Даю пример из сидера который загружает товары.

```php
    $imgUrl = "products/". Str::slug($item['name'])."_".$index.".jpg";
    Storage::disk('public')->put($imgUrl, file_get_contents($item['img']), 'public');
```

Важно чтоюы фото грузились в хранилище в папку products

Внеси в скрипт следующие изменения:

- На каждой итерации в начале проверяем есть ли товар с таким sku, если нету добавляем новый если есть обновляем данные старого товара
- при добавлении нового товар popular = false
